  • git status - показує поточний стан вашого робочого дерева.
  • git branch - Виводить список усіх локальних гілок.
  • git remote - використовується для додавання віддаленого сховища до вашого локального сховища.
  • git add - використовується для додавання файлів в область постановки.
  • git commit - використовується для створення нового коміту.
  • git commit -m - флаг m використовується щоб набрати текстове повідомлення
  • git push - використовується для перенесення змін з вашого локального сховища у віддалений сховище.
  • git push -u origin main - використовується для переміщення поточної гілки у віддалений репозиторій. Прапор -u вказує Git'у встановити зв'язок відстеження між локальною гілкою і віддаленою гілкою.
  • git init - використовується для ініціалізації нового сховища Git.
  • git checkout - використовується для перемикання між гілками.
  • git checkout -b branch-name - використовується для створення нової гілки та перемикання на неї.
